Overview

Role-based views (patient/provider/insurer) with column masking and immutable access logging on a toy EHR dataset.

Stack

How to Run

python src/seed_db.py
python src/demo_access.py --role provider --patient_id 101
# Optional API
python src/app.py  # open http://localhost:5000/patient/101?role=provider
